## Getting Started

Install packages:

```bash
npm install
```

Run the API:

```bash
npm run api
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Run tests:

```bash
npm run test
```
